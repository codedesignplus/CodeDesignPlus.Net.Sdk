## About The Project
`CodeDesignPlus.Net.Kafka` provides a robust framework for integrating Apache Kafka with .NET Core applications. This library simplifies the process of producing and consuming Kafka messages, enabling developers to build scalable and reliable event-driven systems.

### Key Features
1. Message Production: Facilitates easy and efficient production of Kafka messages.
2. Message Consumption: Simplifies the process of consuming Kafka messages with built-in support.
3. Scalability: Designed to handle scalable event-driven architectures.
4. Integration: Seamlessly integrates with other CodeDesignPlus.Net libraries.
5. Telemetry Support: Includes support for OpenTelemetry to monitor and trace Kafka operations.
